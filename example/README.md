# Ustro Vite Example

A minimal Vite example for the current `ustro` package.

## Run

```bash
cd example
bun install
bun run dev
```

## What It Shows

The example currently mounts a simple component so the project stays runnable while the docs and API surface evolve.

As the example grows, it can be expanded to demonstrate `core`, `html`, `modifiers`, `navigator`, and `validator` features in a more interactive way.
